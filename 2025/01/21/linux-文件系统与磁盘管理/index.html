linux-文件的访问控制列表facl
setfacl工具： 设置和修改文件或目录的访问控制列表（ACLs）的命令行工具。 1setfacl -m option:name[:permission] file -m 表示修改现有的 ACL 条目。 option 可以是 u（用户）、g（组）、o（其他）、d（默认）、-（掩码，用于限制通过 ACL 授予的权限）。 name 是用户或组的名称。 permission 是逗号分隔的权限列表，可以是 r（读）、w（写）、x（执行）、R（读，但不允许执行）、W（写，但不允许执行）、X（执行，但不允许写）。
